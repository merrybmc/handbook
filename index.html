SQL 논리 실행 순서

FROM → JOIN → WHERE → GROUP BY → HAVING → SELECT → ORDER BY → LIMIT

WHERE와 HAVING
  • WHERE: 그룹 전, 행 단위 필터
  • HAVING: 그룹 후, 집계 결과 필터
트랜잭션 ACID
  • Atomicity, Consistency, Isolation, Durability
  • COMMIT은 확정, ROLLBACK은 취소
MySQL vs PostgreSQL
  • MySQL: AUTO_INCREMENT, ON DUPLICATE KEY
  • PostgreSQL: IDENTITY, ON CONFLICT, JSONB
